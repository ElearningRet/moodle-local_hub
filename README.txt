Moodle Community Hub Server
===========================

This hub server turns a Moodle site into a Community hub, 
where other sites can share courses as downloads (for use 
as templates) or links (for enrolling in).

Development spec:  
    http://docs.moodle.org/en/Development:Community_hub

Hub Administration
    http://docs.moodle.org/en/Hub_administration



Installation
------------

1. Place this folder in your Moodle as /local/hub 
2. Visit the Moodle admin page (/admin) to install 
3. Configure via the new Hub item in the Admin menu
4. Register your hub with the moodle.org hub directory.
